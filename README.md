# Overview
## Requirements
* FFMPEG
* ffmpeg-python
* numpy
## Installation
Go to your DR `Scripts` folder and copy all the files into `Utility` subfolder.
The scripts folder location is stated in the official API Docs.
## Usage
All of the helper scripts are located under the `common` folder.
To import them in your python script use the import statement just like in the examples.

```from common.cutter import Cutter # Imports a clip cutter```
## Features
* **Clip Cutter**
* **Gaps Deleter**
* **Logs**
